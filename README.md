# 🚀 DonasiYuk

**Pengembangan Platform Web DonasiYuk untuk Meningkatkan Kepedulian Sosial Digital melalui Donasi Barang ke Panti Asuhan**

---

## 📌 Deskripsi Project

**DonasiYuk** merupakan platform web berbasis Laravel yang dirancang untuk meningkatkan kepedulian sosial di era digital dengan memfasilitasi donasi barang kepada panti asuhan.

Melalui platform ini, pengguna dapat menyalurkan barang layak pakai kepada panti asuhan yang membutuhkan secara lebih terstruktur dan tepat sasaran.

Project ini dibuat untuk:

* Mempermudah proses donasi barang secara online
* Menyalurkan bantuan ke panti asuhan secara tepat
* Mengurangi pemborosan barang yang masih layak pakai
* Meningkatkan kepedulian sosial masyarakat
